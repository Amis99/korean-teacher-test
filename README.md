# 칠판 테스트 — 국어 강사 자가진단

학원 국어 강사(초중고 독서논술, 내신, 수능, 인문논술)를 위한 **모바일 전용 자가진단 웹사이트**입니다.

칠판 + 분필 느낌의 심플하고 따뜻한 디자인으로 구현했습니다.

## ✨ 주요 기능

- 28문항 자가진단 테스트 (공통 역량 + 4개 전문 분야 + 성향)
- 즉시 결과 분석 (점수 + 추천 분야 + 성장 조언)
- 강사 유형을 동물/캐릭터로 비유 (토끼, 올빼미, 여우, 창의적인 마법사)
- 강점 · 보완점 · 구체적인 조언 제공
- 완전 모바일 최적화 (손가락 터치에 최적화)

## 🚀 GitHub Pages 배포 방법

1. 이 폴더 전체를 새 GitHub 저장소에 업로드하세요.
2. 저장소 설정 → Pages → Source를 `main` 브랜치 / `(root)` 로 선택
3. 배포 완료! (보통 1~2분 소요)

## 🎨 디자인 특징

- 진짜 칠판 질감 배경
- 분필 손글씨 스타일 폰트 (Caveat)
- 노란 분필 포인트 컬러
- 심플하고 직관적인 UI
- 모든 버튼과 카드가 모바일에서 편안하게 터치 가능

## 💡 사용 팁

- 테스트는 약 5~7분 정도 소요됩니다.
- 솔직하게 답변할수록 더 정확한 결과가 나옵니다.
- 결과 페이지에서 강점/보완점과 추천 분야를 확인하세요.
